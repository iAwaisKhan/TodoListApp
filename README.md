# To-Do List Application 

## Description
A simple to-do app to add, complete, and delete tasks with smooth animations for easy task management.

## Features
- Add new tasks 
- Mark tasks as complete 
- Delete tasks 
- Smooth transitions and animations
- Task filtering and search 

## Installation
1. Clone or download the project 
2. Open `index.html` in your browser 

## Usage
- Type a task into the input box and press Enter or click Add 
- Click the checkbox to mark a task complete 
- Click the delete button to remove a task 
- Use search and filter options for task organization
